Using Ajax control and JQuery in .Net


I have the ASP.Net page,in that i have one button and on click of that button calling an alert msg.In have achieved this using jQuery.
However,there is much simpler way,like on click event of that button i would have called a function/a alert box as in my case.

Now I am using Microsoft Ajax framework,utilizing the UpdatePanel control.
The reason why I do that is that certain elements are created on the page based on some server-side logic.Also it could be written by using jQuery but i have tried it that with updatePanel.
Also I had another button inside the very same updatePanel and onclick of that Autopost is happening and the dropdownlist gets filled.

Now the problem which I was facing once I have a postback on the updatePanel the jQuery events stops working.
I actually was expecting this, since the "postback" is not really a new postback so my code in document.ready that binds the events won't be fired again.
So was left with the problem of rebinding my controls after the UpdatePanel is done updating the DOM.

So i tried for partial load function and it worked....


function pageLoad(sender, args)
{
   if (args.get_isPartialLoad())
   {
       //Specific code for partial postbacks can go in here.
   }
}

For writing partial class i hav taken help from this link:
http://www.itfunda.com/Howto

Another solution to the above problem was simply kept the other button and the dropdown list outside the update template that was using
jQuery,as whenevr post back is happened the attachment to that content is lost.so have to reattach and specify the DOM attachment.
Or use the partial class instead.
For info on updatepanel in Ajax I had refered this link:
 http://updatepanelplugin.codeplex.com
